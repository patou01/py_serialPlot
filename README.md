# py_serialPlot

A simple serial plotter written in Python, works on Python 3.7

# Features
* Takes any number of inputs, all formulated in line "y1, y2, y3 ... yn\n".
* Can use the first input as x axis


# libraries and versions
* Matplotlib 3.3.2
* numpy 1.19
* pySerial 3.4

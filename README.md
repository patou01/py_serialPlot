# py_serialPlot

A simple serial plotter written in Python, works on Python 2.7.12.

# Features
* Takes any number of inputs, all formulated in line "y1 y2 y3 ... yn\n".
* Can use the first input as x axis


# libraries and versions
* Matplotlib V1.5.1
* numpy 1.11.0
* pySerial 3.0.1
